# NeoPulse - Neonatal Vital Signs Monitor

₹1,200 wearable monitor for preterm newborns. Saves lives in rural Indian hospitals.

## Hardware
- Wearable: Arduino Nano + MAX30102 + DS18B20 + MPU6050
- Hub: Arduino Uno + OLED + RGB LED + Buzzer

## Code
- `neopulse_wearable.ino` → Baby sock firmware
- `neopulse_hub.ino` → Nurse station hub

## Demo
Green/Yellow/Red alerts + buzzer for 5-10 min early warning.
